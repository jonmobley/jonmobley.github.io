# GitHub Pages Root Domain

This repository is for hosting the `apple-app-site-association` file at the root domain `jonmobley.github.io`.

## Setup Instructions

1. Create a new repository on GitHub named: `jonmobley.github.io`
2. Upload the contents of this folder to that repository
3. Enable GitHub Pages in the repository settings (it will automatically use the `main` branch)
4. The file will be accessible at: `https://jonmobley.github.io/.well-known/apple-app-site-association`

## Important

- This repository must be named exactly `jonmobley.github.io` (your username + `.github.io`)
- The `.well-known` folder and `.nojekyll` file must be in the root of the repository
- After uploading, wait a few minutes for GitHub Pages to deploy
